# YoRHaUI

这是一个适用于Vue3的UI组件库，其所有视觉效果和风格设计都源于NieR: Automata这款游戏内的UI系统。

> [!INFO] 这并不是一个通用型或者企业级的UI库，而是偏向"游戏设计语言"的组件库。这意味着它并不会拥有太多复杂性。你可以用它给一些简单Vue项目设计出一个漂亮的后台，或者甚至可以将其应用于真正复杂的后台系统中，谁知道呢🤷？

## 开发日志

这个项目涉及的技术:

1. 框架：Vue3
2. 语言：Typescript
3. 打包：Vite + Rollup
4. 文档：VitePress

这是一个可能逐渐复杂到可怕的项目，需要一定的工具去约束：

1. husky：提交的约束
2. prettie：代码风格
3. eslint：规范检测
